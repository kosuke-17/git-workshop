# Git 基礎演習 エクササイズ 1

## 目的

この演習では、Git の基本的なコマンドを実践的に学びます。簡単な Web ページを作成・修正しながら、Git でのバージョン管理の流れを体験します。

## 演習の流れ

### 準備

1. このディレクトリで新しい Git リポジトリを初期化します
2. `Challenge`フォルダ内の HTML と CSS ファイルを使って演習を進めます
3. 各ステップで Git コマンドを実行し、変更を記録していきます

### 学習内容

- `git status`: 現在の状態確認
- `git add`: ファイルをステージング
- `git commit`: 変更をコミット
- `git log`: コミット履歴の確認
- `git diff`: 変更内容の確認
- `git restore`: 変更の取り消し

## 完了後

すべてのチェックリストを完了したら、`Answer`フォルダの解説を確認して理解を深めましょう。

## ヒント

- コマンドがわからなくなったら`git --help`や`git <コマンド> --help`で確認できます
- `git status`は何度実行しても問題ありません。迷ったら実行しましょう
- コミットメッセージは変更内容がわかるように具体的に書きましょう
